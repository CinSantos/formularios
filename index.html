a. Na tag input com atributo type="text", para que serve o atributo "placeholder"?
O atributo placeholder é uma pequena dica, uma pequena frase, uma palavra, que tem o intuito de ajudar o usuário a entender como ele deve preencher aquele formulário.

b.Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?
O atributo checked especifica que o botão "radio" é selecionado por default, ou seja, por padrão.

c.Na tag input para que serve o atributo "value"?
Este atributo especifica que o usuário deve preencher o campo com um valor.

d.Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez? Para que somente um dos botões de rádio seja marcado, os mesmo devem pertencer ao mesmo grupo, o que pode ser atribuindo ao atributo value o mesmo nome.

e. Para que serve a tag label? Para que serve o atributo "for" nessa tag?
A tag label define um rótulo para muitos elementos de formulário. O for, nesse sentido, permite referenciar o objeto e facilitar a usabilidade. É útil para usuários de leitores de tela, porque o leitor de tela lerá em voz alta o rótulo quando o usuário focar no elemento de entrada; também ajuda os usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de opção ou caixas de seleção). O atributo label deve ser igual ao atributo id do input para ligá-los.

f.Na tag option para que serve o atributo "selected"?
Os elementos option definem uma opção que pode ser selecionada. Por padrão, o primeiro item na lista suspensa é selecionado. Para definir uma opção pré-selecionada, adicione o atributo selecionado à opção.

g. Nas tags de formulário para que serve o atributo "required"?
Required refere-se a uma restrição de entrada comum. Especifica que um campo de entrada é obrigatório (deve ser preenchido)

h. Na tag form para que serve o atributo "action"? O atributo action define a ação a ser executada quando o formulário for enviado. Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.

i.Na tag form para que serve o atributo "method"?
O atributo method especifica como enviar dados de formulário (os dados de formulário são enviados para a página especificada no atributo action). Os dados do formulário podem ser enviados como variáveis ​​de URL (com method="get") ou como transação de postagem HTTP (com method="post").

j. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
Para que os dados não sejam reproduzidos na barra de endereços do navegador, é necessário utilizar o method post, haja vista que get permite a visualização integral do conteúdo.

k. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
O method post permite realizar uploads de arquivos de texto e binários.
